// how it works

Six stages, one pass

01 · LISTEN

Transcribe

Word-level timestamps from the source audio.

via faster-whisper
02 · SELECT

Score

The model picks moments and scores each 0–100.

via Qwen3 / Ollama
03 · TRIM

Cut silence

Dead air between words gets removed automatically.

timestamp math
04 · REFRAME

Track

Speaker tracked into 9:16 with a smoothed camera.

via OpenCV YuNet
05 · CAPTION

Burn in

Karaoke captions with a persistent hook headline.

ASS + libass
06 · EXPORT

Encode

h264 / yuv420p / aac — plays anywhere, GPU-accelerated.

via ffmpeg

// what it does

Built for the edit, not the demo

[ layout ]

Three layouts

Single speaker, speaker-over-gameplay, or automatic Twitch facecam separation.

3fill · split · stream
[ scoring ]

Virality score

Every clip is ranked 0–100 with a hook headline generated alongside it.

0—100per clip
[ editing ]

Silence trim

Dead air between words is cut automatically — no manual scrubbing.

autoon by default
[ motion ]

Punch-in zoom

A subtle camera tighten on emphasized words, tracked to the speaker.

8%default punch
[ b-roll ]

Auto B-roll

Optional stock-footage cutaways on keyword moments, via a free Pexels key.

opt-inno key, no cutaway
[ iterate ]

Regenerate one clip

Re-render a single clip with new settings without redoing the slow stages.

↻reuses transcript
